Mobile Navigation Test

This page tests the enhanced mobile navigation system. To test:

  • Desktop Dropdowns: Hover over "Case Status" and "More" to see dropdown menus
  • Mobile Testing: Resize your browser window to mobile size (below 900px width)
  • Mobile Menu: Click the hamburger menu button (☰) in the top right
  • Mobile Dropdowns: Click on "Case Status" and "More" to expand/collapse
  • Close Menu: Click the hamburger button again or click the overlay
  • Swipe Gestures: Test swipe gestures on mobile devices

Expected behavior:

  • Desktop: Dropdowns should appear on hover
  • Mobile: Hamburger menu should animate when opening/closing
  • Mobile: Dropdown menus should expand/collapse smoothly
  • Mobile: Menu should close when clicking outside or pressing ESC
